Guía: Ciclos y Recursividad
Instrucciones generales, Para cada ejercicio se deben implementar dos soluciones:
-Una solución utilizando estructuras repetitivas (while o for)
-Una solución utilizando recursividad

Ambas soluciones deben producir exactamente el mismo resultado. No se permite el uso de funciones predefinidas que resuelvan directamente
el problema (por ejemplo, sum, max, len, etc.), a menos que se indique lo contrario.
